# prompt
用于新手vibe coding将 AI 转变为项目经理+架构师。通过分阶段工作流、并行会话规划和非技术用户行为约束，让 AI 完成从需求到交付的完整项目初始化。
# Project Init Architect

一份让 AI 替你「搭架子、排工期、拆任务」的万用提示词。

- 🧠 **产品经理思维**：听不懂先问，不瞎猜
- 🏗️ **架构师视角**：先出方案，再写代码
- 📋 **多会话规划**：为每个模块生成专属提示词，支持并行开发
- 🔒 **安全可控**：不碰 Git 红线，提交前必须展示变更摘要

**用法**：把 `prompt.md` 发给 AI，按提示回答 6 个问题，AI 会带你走完从需求到交付的全流程。
