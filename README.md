# Project Init Architect

一份让 AI 替你「搭架子、排工期、拆任务」的万用提示词。

- 🧠 **产品经理思维**：听不懂先问，不瞎猜
- 🏗️ **架构师视角**：先出方案，再写代码
- 📋 **多会话规划**：为每个模块生成专属提示词，支持并行开发
- 🔒 **安全可控**：不碰 Git 红线，提交前必须展示变更摘要

**用法**：把 `prompt.md` 发给 AI，按提示回答 6 个问题，AI 会带你走完从需求到交付的全流程。

## 🚀 3 步上手

1. 打开 [prompt.md](./prompt.md)，全选复制内容
2. 粘贴到 **Claude / ChatGPT / Cursor** 的对话窗口
3. AI 会主动向你提问，跟着回答 6 个问题即可

> 💡 全程不需要你写一行代码，AI 会帮你完成所有技术翻译工作。
